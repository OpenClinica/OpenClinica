3.5 CRFs Help

This section allows you to view all the Case Report Forms (CRFs) that are available for use in your studies and sites. All CRFs are ordered through their CRF Name. You can choose to reorder the list of CRFs by toggling the ascending or descending arrow located next to the CRF Name header. Similarly, you can also reorder the CRF list using the CRF’s Date Updated, Last Updated By, Versions_OID, or Date Created, and toggle their respective ascending or descending arrows.

With the Find function, located at the top of the CRF list, you can search for a set of CRFs based on their CRF Name.

In addition to viewing all the CRFs, you are also presented with links to download a Blank CRF Template, to go to the OpenClinica CRF Library as well as to Create a New CRF.

The Manage Case Report Forms (CRF) matrix displays the following columns:

1. CRF Name This cell indicates the CRF Name.
2. Date Updated This cell indicates the date the CRF was last updated.
3. Last Updated By This cell indicates the user who last updated the CRF.
4. CRF_OID This cell indicates the specific Object Identifier (OID) for each CRF.
5. Versions This cell indicates the specific version of a CRF. Note: that each CRF has at least two versions associated with it. The “(original)” is the original version of the CRF that was created the first time its particular CRF Name was used. The version(s) listed below it is/are the version(s) that is/are currently in use. Each non-original version can be continually replaced with updates to the CRF. However, the original version (with the CRF Name associated with it) will always remain in the OpenClinica system.
6. Version_OID This cell indicates the specific Object Identifier (OID) for each version of a CRF.
7. Date Created This cell indicates the date the CRF was created.
8. Owner This cell indicates the owner who originally created the CRF.
9. Status This cell indicates the status of each CRF. It could be available or locked.
10. Download This cell displayes the Download SpreadSheet icon to downlaod the specific version of the CRF.
CRF Versions which have no downloadable file are indicated with “N/A”.
11. Actions (depending on role) View

Clicking on the View icon will bring you to the View CRF Details page.

Edit (only available for the original CRF version)

Clicking on the Edit icon will allow you to edit the Name and Description of original CRF.

Remove

To remove a CRF from the study click the Remove icon.

Create New Version (only available for the original CRF version)

Clicking on the Create New Version icon will allow you to add a new updated excel spreadsheet. Doing so will still keep the “(original)” and the other versions intact.

Delete (only available for CRF versions)

If you want to delete a particulary CRF version click on the Delete icon. Note: You can only delete CRF versions that have no associated study event associated study event definitions.
