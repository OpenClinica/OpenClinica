5.3 Users Help

This section allows you to view all the users that are a part of your current study. All users are ordered through their User Name. You can choose to reorder the list of users by toggling the ascending or descending arrow located next to the User Name header. Similarly, you can also reorder the subject list using the user’s First Name, Last Name, Role, Study Name, or Status, and toggle their respective ascending or descending arrows.

The Find function, located at the top of the user list, allows you to search for a set of users based on their User Name, First Name, Last Name, Role or Study Name.

1. User Name This cell indicates the user name of each user.
2. First Name This cell indicates the first name of each user.
3. Last Name This cell indicates the last name of each user.
4. Role This cell indicates the current role of each user.
5. Study Name This cell indicates the study assigned to each user.
6. Status This cell indicates the status of each user.
7. Actions View

Clicking on the View icon will allow you to see all the details associated with the particular user.

Set Role

Clicking on the Set Role icon will allow you to edit the Role of the particular user.

Remove Role

Clicking on the Remove Role icon will allow you to remove the particular user. From this point on, the user will no longer have access to the specific study, unless the user is restored.

Restore

Clicking on the Restore icon will allow you to restore the particular user. From this point on, the user will have access to the specific study, unless the user is removed.

In addition to viewing all the users, you are also presented with a link to Assign New User to Current Study.
