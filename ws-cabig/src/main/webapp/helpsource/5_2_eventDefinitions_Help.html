5.2 Event Definitions Help

This section allows you to view all the Study Events that are a part of your current study. All Study Events are ordered through their Name. You can choose to reorder the list of events by toggling the ascending or descending arrow located next to the Name header. Similarly, you can also reorder the Study Event list using the event’s Repeating, Type, Populated, or Date Updated, and toggle their respective ascending or descending arrows.

The Find function, located at the top of the Study Event list, allows you to search for a set of events based on their Name or Type.

If you click on the Show More link in the top of the table you can also see columns for the name of the CRFs included in this event as well as their Default Version.

CRFs: This cell displays all CRFs in this Study Event.
Default Version: This cell indicates the default version of the CRF.

In addition to viewing all the Study Events, you are also presented with a link to Create a New Study Event Definition.

The Study Event Definition matrix displays the following columns by default:

1. Name This cell indicates the Name of each Study Event.
2. OID This cell indicates the object identifier (OID) of each Study Event.
3. Repeating This cell indicates if the particular Study Event is repeating or non-repeating.
4. Type This cell indicates the type of each Study Event. It could be Scheduled, Unscheduled, or Common.
5. Category This cell indicates the category under which each Study Event falls under.
6. Populated This cell indicates if the particular Study Event is populated with subjects.
7. Date Created This cell indicates the date the Study Event was created.
8. Date Updated This cell indicates the date the Study Event was last updated.
9. Actions View

Clicking on the View icon will allow you to see all the details associated with the particular Study Event.

Edit

Clicking on the Edit icon will allow you to edit all the properties of the Study Event, including the Name, Description, Repeating, Type, and Category. You can also review your CRF preferences and add/remove CRFs.

Remove

Clicking on the Remove icon will allow you to remove the particular Study Event. From this point on, only the Administrator can access and reverse this change if necessary.

Restore

Clicking on the Restore icon will allow you to restore the particular Study Event. From this point on, the event will be assigned back to the specific study, unless the event is removed.
