6.2.2 Audit User Activity Help

This section is designed to list all user activities. The matrix provides you with the following information:

User Name: This cell indicates the user name.
Login Attempt Date: This is the server date and time when the action was performed.
Login Status: This cell indicates the status of the action. If it was successfull or failing.
Action: This cell provides you with the View icon to go to the View User Account page.

In the top of the table you can use the green arrows to navigate through the pages.
With the drop-down menu next to the arrows you can choose how many user activities should be displayed (15, 50, or 100).
To export the audit log as CSV or XLS click on the appropriate icon.
The Filter and Clear icons indicate the filter function. Just type in a User Name, Date, or Login Status in the fields below and hit the Filter icon.
