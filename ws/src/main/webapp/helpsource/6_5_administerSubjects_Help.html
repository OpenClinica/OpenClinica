6.5 Subjects Help

This section allows you to view all the subjects that are available to you across all studies. You can choose to reorder the list of subjects by clicking on a column title. The ascending or descending arrow located next to the column header indicates the sorting.

In the top of the table you can use the green arrows to navigate through the pages. If you have less than 15 subjects in your study the arrows are greyed out, which means that there is only one page to display.
With the drop-down menu next to the arrows you can choose how many subjects should be displayed on one page (15, 25, or 50).

The Administer Subjects matrix displays the following columns:

1. Person ID This cell displays the Person ID, which can be used to link subjects from different studies to a single individual.
2. Protocol-Study Subject IDs This cell displays the study (short protocol ID) and study subject ID together. It is not available for sorting.
3. Sex This cell indicates the sex of each subject.
4. Date Created This cell indicates the date the subject was created.
5. Owner This cell indicates the owner who originally created the subject.
6. Date Updated This cell indicates the date the subject was last updated.
7. Last Updated By This cell indicates the user who last updated the subject.
8. Status This cell indicates the status of each subject. It could be available, locked, removed, or auto-removed.
9. Actions View

Clicking on the View icon will allow you to see all the details associated with the particular subject.

Edit

Clicking on the Edit icon will allow you to edit the subject’s Person ID, Gender, and Date of Birth.

Remove

Clicking on the Remove icon will allow you to remove the particular subject. From this point on, only the Administrator can access and reverse this change if necessary.

Restore

Clicking on the Restore icon will allow you to restore the particular subject. From this point on, the subject will be available for studies, unless the subject is removed.

You can filter by entering free text or by selecting a pre-defined choice from the drop-down menu, in the grey boxes underneath each column title. To apply the filter select the Apply Filter link.
To reset the filter click on the Clear Filter link next to it.
